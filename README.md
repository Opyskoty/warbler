# Warbler

(built by Olivia and Diana)

Demo: https://olivia-and-diana.herokuapp.com/

A twitter knockoff written in Python using Flask and SQLAlchemy.

## How did we build this?:

- Flask
- SQLAlchemy
- PostgreSQL
- WTForms
- Bcrypt
- Jinja

## What can you do?:

- Signup/Login/Logout
- Liking/Disliking
- Follow other users
